# Proyecto de Regresión Lineal: Consumo de Combustible

Este proyecto aplica un modelo de regresión lineal para predecir el consumo de combustible basado en las emisiones de CO, el tamaño del motor y el número de cilindros.

## Estructura del Proyecto

- *data/*: Carpeta que contiene los datos del proyecto.
  - Fuel.csv: Archivo de datos utilizado para el análisis.
- *scripts/*: Carpeta que contiene los scripts de código.
  - main.py: Script principal que ejecuta el análisis de regresión lineal.
- *.gitignore*: Archivo que especifica qué archivos y carpetas deben ser ignorados por Git.
- *README.md*: Descripción del proyecto.

## Requisitos

- Python 3.x
- Bibliotecas:
  - pandas
  - matplotlib
  - scikit-learn
  - numpy
